# Hospital Management System

## Project Description
The **Hospital Management System (HMS)** is a web-based application designed to streamline the management of hospitals and clinics. It helps hospital staff manage patient records.
---

## Features
- **Hospital Management:** Add, update, delete, and search hospital and patient records. 
---

## Technologies Used
- **Backend:** Java, Spring Boot, Hibernate/JPA  
- **Database:** MySQL
- **Tools:** Git, Maven, Postman for API testing
